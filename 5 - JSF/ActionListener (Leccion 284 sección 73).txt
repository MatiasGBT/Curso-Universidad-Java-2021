Action listener en JSF:
JSF provee dos maneras de configurar la detección de eventos cuando un usuario ejecuta una acción de tipo submit, ya
sea agregando el atributo actionListener al componente JSF o como un elemento interno en el componente.

Algunos componentes de este tipo son h:commandButton y h:commandLink.
<h:commandLink actionListener="#{vacanteForm.enviar}"/>

Una vez definido el atributo o el elemento interno, se debe agregar el método action en el Managed Bean que procesa
la petición de la Vista:
public void enviar(ActionEvent actionEvent) {...}

Este tipo de componentes permite procesar formularios de las páginas web.
